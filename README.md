# 📚 Book Revelation 3D

Интерактивный 3D модуль с вращающейся книгой стоической философии для интеграции на сайт (Tilda, WordPress и др.)

**Название на языках:**
- 🇬🇧 English: **Book Revelation 3D**
- 🇷🇺 Русский: **Откровение Книги 3D**
- 🇮🇱 עברית: **גילוי הספר 3D**
- 🇨🇳 中文：书籍启示 3D

## ✨ Возможности

- 🎨 **3D Книга** с детализированной обложкой и текстом
- 🔄 **Плавное вращение** с эффектом парения
- 💡 **Интерактивность** — клик для паузы/возобновления вращения
- 💬 **40 цитат** из 5 книг великих мыслителей
- ✨ **Анимированная подсветка** активной цитаты
- 📱 **Адаптивный дизайн** — работает на мобильных
- 🌟 **Частицы и эффекты свечения**
- 🎛️ **Единое главное меню** с навигацией по разделам
- 🎨 **6 тем оформления** (Тёмная, Светлая, Синяя, Фиолетовая, Атмосферная, Расслабляющая)
- ⭐ **Избранные цитаты** с экспортом/импортом
- 🧘 **Zen-режим** для погружения
- 🔍 **Поиск и фильтры** по цитатам

## 🚀 Быстрый старт

```bash
# Установка зависимостей
bun install

# Запуск в режиме разработки
bun run dev

# Сборка для продакшена
bun run build
```

## 🔌 Интеграция в Tilda

После деплоя на Vercel/Netlify, вставьте в HTML-блок Tilda:

```html
<iframe 
  src="https://your-domain.vercel.app" 
  width="100%" 
  height="600px" 
  frameborder="0" 
  allowfullscreen
  style="border: none; overflow: hidden; border-radius: 12px;"
></iframe>
```

### Рекомендации для Tilda:

- **Минимальная высота:** **500px**
- **Ширина:** **100%** (контейнер должен быть широким)
- Работает в **Zero Block** и стандартных блоках

## 🎨 Кастомизация

### Изменить цитаты

Отредактируйте массив `quotes` в файле `src/data/books.ts`:

```typescript
quotes: [
  { text: "Ваша цитата...", author: "Автор", era: "Годы жизни" },
  // ...
]
```

### Изменить цвета

**Основные цвета (золотой акцент):**

- Primary: `#d4af37`
- Background: `#0a0a15`

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Пробел` | Пауза/вращение книги |
| `Z` | Zen-режим (скрыть UI) |
| `Esc` | Выход из Zen-режима / Закрыть меню |
| `↑` `↓` | Навигация по цитатам |
| `+` `-` | Приблизить/отдалить камеру |
| `0` | Сброс масштаба камеры |
