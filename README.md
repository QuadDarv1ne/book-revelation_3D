# 📚 Book Revelation 3D

Интерактивный 3D модуль с вращающейся книгой стоической философии для интеграции на сайт (Tilda, WordPress и др.)

![Book Revelation 3D](https://img.shields.io/badge/Next.js-15-black?style=flat-square&logo=next.js)
![Three.js](https://img.shields.io/badge/Three.js-r148-orange?style=flat-square&logo=three.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=flat-square&logo=typescript)

## ✨ Возможности

- 🎨 **3D Книга** с детализированной обложкой и текстом
- 🔄 **Плавное вращение** с эффектом парения
- 💡 **Интерактивность** — клик для паузы/возобновления вращения
- 💬 **8 цитат** Марка Аврелия и Эпиктета
- ✨ **Анимированная подсветка** активной цитаты
- 📱 **Адаптивный дизайн** — работает на мобильных
- 🌟 **Частицы и эффекты свечения**

## 🚀 Быстрый старт

```bash
# Установка зависимостей
bun install

# Запуск в режиме разработки
bun run dev

# Сборка для продакшена
bun run build
```

## 🔌 Интеграция в Tilda

После деплоя на Vercel/Netlify, вставьте в HTML-блок Tilda:

```html
<iframe 
  src="https://your-domain.vercel.app" 
  width="100%" 
  height="600px" 
  frameborder="0" 
  allowfullscreen
  style="border: none; overflow: hidden; border-radius: 12px;"
></iframe>
```

### Рекомендации для Tilda:

- **Минимальная высота:** **500px**
- **Ширина:** **100%** (контейнер должен быть широким)
- Работает в **Zero Block** и стандартных блоках

## 🎨 Кастомизация

### Изменить цитаты

Отредактируйте массив `quotes` в файле `src/app/page.tsx`:

```typescript
const quotes = [
  { text: "Ваша цитата...", author: "Автор", era: "Годы жизни" },
  // ...
]
```

### Изменить название книги

Отредактируйте функцию `createBookCoverTexture()` для изменения обложки.

### Изменить цвета

**Основные цвета (золотой акцент):**

- Primary: `#d4af37`
- Background: `#0a0a15`
